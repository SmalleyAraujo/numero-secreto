# 🎯 Número Secreto

Projeto de um jogo simples onde o usuário deve adivinhar um **número secreto gerado aleatoriamente**. O sistema informa se o número digitado é **maior ou menor** até o jogador acertar.

## 🎮 Como jogar

1. Digite um número.
2. O sistema informará se o número secreto é **maior ou menor**.
3. Continue tentando até acertar.
